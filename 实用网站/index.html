实用网站

发表于 2020-02-29 分类于 工具
本文字数： 1.1k 阅读时长 ≈ 1 分钟
常用网址
GitHub

GitHub

  • 存放自己的一些Demos，源代码
  • 查找一些优秀的项目或者学习资料
  • 使用githubPage构建静态博客
清华镜像

清华镜像

  • 一些国外开源的工具或者资源，都有镜像下载，国内访问速度快
  • 各种Linux发行版镜像下载以及对应的软件源
  • CTAN的镜像，如TeX & LaTeX 镜像下载
  • 可以指定Python的pip包管理下载源为清华镜像，install package时很快
  • 有Anaconda镜像支持
源代码高亮显示

planetb

  • 源代码直接复制粘贴到Word中，有高亮显示和行号
  • 支持C/C++、Java、Python、C#、JavaScript、PHP、HTML、CSS、Ruby、SQL
开源项目

sourceforge

  • 可以下载开源项目的源代码
  • 查找一些开源的项目
图片格式转换

cloudconvert

  • 支持矢量图格式SVG和EPS在线相互转换
Windows

OS
MSDN

  • 提供Win7、Win8、Win10 系统下载
  • 第二个提供Microsoft的各类工具
职业信息网络

onetonline

  • 美国一个职业信息网络，包含数百万的职业定义
  • 学生或者求职者可以查看对应职业所需的技能要求以及日常工作
免费PPT模板

PPT

  • 可以下载免费的PPT模板
  • 不需要的账号登录
Ubuntu18安装教程

Ubuntu

  • 比较详细的Ubuntu18下载、安装、配置源 的教程
百度谷歌学术

百度学术
谷歌学术镜像

  • 常用于检索一些学术文章、论文等等
人工智能AI
TensorFlow

TensorFlow

  • 安装TensorFlow官网教程
  • 学习TensorFlow
  • TensorFlow相关资源
sklearn官方

scikit-learn

  • 了解、学习机器学习库sklearn
  • 学习机器学习的几种类型：分类、回归、线性、降维
  • 学习对数据的预处理、特征工程、模型选择
kaggle数据集

kaggle

  • 可以直接检索到 kaggle 的数据集
  • 可以下载 kaggle 数据集
LaTeX学习
LaTeX工作室

LaTeXstudio

  • 下载、安装、使用和学习 LaTeX 的教程以及资源
  • 相关的LaTeX、Beamer模板
Beamer

Beamer

  • Beamer的各种主题以及对应的颜色主题预览
LaTeX模板

latextemplates

  • LaTeX 的一些模板下载
arxiv

arxiv

  • arXiv是一個收集物理学、数学、计算机科学、生物学与数理经济学的论文预印本的网站
  • 可以下载相应的学术期刊型LaTeX模板，进行论文的写作
